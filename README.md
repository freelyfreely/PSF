# Prep Set Forget

A tool for Hawaiʻi Island. It reads the real rain and elevation at any point on the island and says
which food plants fit that ground, in the order you would plant them:

**Prep** — what the site needs · **Set** — how it goes in · **Forget** — what happens after.

One HTML file, no build step, no account, no tracking. It works on an old phone on a slow rural
connection, and the flyers survive a photocopier.

The plant list is the delivery mechanism. The actual product is an inner succession — the shift from
seeing land as inert substrate to seeing it as a living community that is already going somewhere.

Nothing on the map is drawn by hand: the coastline is US Census, the rain bands are the Rainfall
Atlas of Hawaiʻi, elevation is USGS 3DEP. Temperature is derived from elevation and labeled derived
everywhere it appears. "Nothing on this list will feed you here" is a designed answer, not an error.
